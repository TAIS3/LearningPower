# LearningPower
##auto.js脚本自动完成学习强国文章阅读与视频播放任务

学习强国版本 2.17.1
主要功能是完成文章阅读与视频播放,25分到手.

脚本开始后可以选择任务"文章阅读""视频播放""全部任务"

文章主要选择为"综合"栏目里面的["人民日报","新华社","求是网"],保证综合栏目前三个是这三个订阅

视频播放选的是电视的播放.采用今天的星期数来匹配.
